JavaScript API

The widget exposes a global API for programmatic control:

// Open the widget
window.FeeDLooPWidget.open();

// Close the widget
window.FeeDLooPWidget.close();

// Reset the form
window.FeeDLooPWidget.reset();

// Get widget state
const state = window.FeeDLooPWidget.getState();

// Check version
console.log(window.FeeDLooPWidget.version);

Configuration Options

Customize the widget behavior with data attributes:

<script
  src="feedloop-widget.js"
  data-project-key="YOUR_PROJECT_KEY"
  data-position="bottom-right"
  data-theme="default"
></script>
